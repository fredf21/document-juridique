# document-juridique
- **Analyse de Risques** : Identifier les risques potentiels cachés dans les clauses d'un contrat.
- **Conseil Juridique Contextuel** : Fournir des conseils sur les actions à entreprendre face à un cas juridique spécifique.
- **Documents Nécessaires** : Lister et aider à préparer les documents requis pour une situation juridique donnée.
- **Comparaison de Documents** : Analyser les différences entre deux versions d'un contrat (idée proactive).
